# DaddyPower Brand Guide — Kurzreferenz für Entwicklung

## Farben

| Farbe | HEX | CSS Variable | Verwendung |
|---|---|---|---|
| Amber Primary | #BA7517 | kraft-accent | CTAs, Highlights, Logo "POWER" |
| Amber Light | #FAEEDA | kraft-accent-light | Hintergrund-Fills, Cards |
| Amber Dark | #633806 | kraft-accent-dark | Text auf hellem Amber-BG |
| Charcoal | #1B1B19 | kraft-dark | Primärtext, Headlines, Logo "DADDY" |
| Offwhite | #F5F3ED | kraft-offwhite | Seitenhintergrund |
| Warm Gray | #B4B2A9 | kraft-border | Borders, Trennlinien, inaktiv |
| Warm Gray Dark | #5F5E5A | kraft-muted | Body-Text, Sekundärtext |
| Weiss | #FFFFFF | — | Cards, Panels |

## Typografie
- Font: Inter (Web) / Arial (Dokumente)
- Headlines: Bold 700
- Body: Regular 400, 16px
- Keine Serif-Fonts

## Tonalität
- Kraftvoll, kameradschaftlich, ehrlich, wissenschaftlich fundiert
- Du-Form immer
- Kurze Sätze, keine Konjunktive
- Nie zynisch über Familie/Partnerinnen

## Logo
- Blitz-Kreis-Symbol + "DADDY / POWER" Wordmark
- DADDY in Charcoal, POWER in Amber
- Amber-Trennlinie zwischen den Zeilen
